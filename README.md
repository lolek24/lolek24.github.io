# lolek24.github.io

Personal portfolio website for **Tomasz Dygas** — SAP Developer, Solution Architect & DevOps Specialist based in Warsaw, Poland.

Live site: [https://lolek24.github.io](https://lolek24.github.io)

## Tech Stack

- **Jekyll** — static site generator
- **GitHub Pages** — hosting & deployment
- **just-the-docs** — remote theme
- **Font Awesome** — icons (CDN)

## Pages

- **Home** (`index.md`) — hero banner, feature cards, stats, about, services, contact
- **Portfolio** (`portfolio.md`) — projects, skills grid, experience timeline, competencies
- **Blog** (`blog.md`) — blog listing (under construction)

## Features

- EN/PL language switcher (client-side, `localStorage`)
- Responsive design
- Custom CSS design tokens (minimalist palette)

## Deployment

The site is automatically built and deployed by GitHub Pages on push to `main`.
